What API are you using?

We decided to use a weather API from metaweather.com.

A sketch or mock up of your design

We plan to have a heading that has a different message that is determined if the data is loaded or failed to load. There should be two different search functions, one that searches by a location given, and the other that searches every location that fits a certain condition. For each location, there should be a display of all the useful data (high/low temp, current temp, humidity, visibility, wind speed, etc.) For the conditions page, we want it to display a list of locations that, when clicked, take you to that location display.
